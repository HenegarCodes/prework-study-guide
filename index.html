HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • src in the img tag contains the location of media
  • alt contains the alternate caption or description of the media if it isn't working
  • h1 is the largest heading we can have and should only be used once

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Assigning colors in CSS can be done in many ways, including by semantic references such as gray or blue. We can also refer to a color by its hexadecimal code, or hex #.
  • The box-shadow CSS property adds shadow effects around an element's frame. You can set multiple effects separated by commas. A box shadow is described by X and Y offsets relative to the element, blur and spread radius, and color.

Git

  • git status: checks what branch we are currently on
  • cd ___: searches and puts us into that folder/directory
  • git checlout -b branch-name: creates a new branch and switches to it
  • git add: Moves changes from the working directory to the staging area. This gives the opportuity to prepare a snapshot before committing it to the official history git add -A: is used to command and commit all files in the working branch git commit -m "__": is used to associate a message with our commit git pull origin main: pulls the base branch through terminal to check changes are actually made
  • git commit: tkae the staged snapshot and commits it to the project history. Combined with the git add, this defines the basic workflow for all gits.
  • git fetch: Fetching downloads a branch from another repository, along with all of its associated commits and files. But, it doesn't try to integrate anything into your local repository. This gives you a chance to inspect changes before merging them with your project.
  • code . : opens up VSCode

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
